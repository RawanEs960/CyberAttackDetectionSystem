# CyberAttackDetectionSystem

## Project Overview
This project is a web-based Intrusion Detection System using Machine Learning to classify network traffic as normal or attack.

## Dataset
CICIDS dataset

## Model
- XGBoost classifier
- Data preprocessing and feature engineering applied
- Model evaluation: Precision, Recall, Confusion Matrix

## Web Interface
- Built with Flask / FastAPI
- Displays real-time predictions and alerts

## How to Run
1. Clone the repository
2. Install required packages: `pip install -r requirements.txt`
3. Run the web app: `python app/app.py`
